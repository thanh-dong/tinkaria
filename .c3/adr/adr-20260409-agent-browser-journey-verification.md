## Goal

[ASSUMED] Add a first practical agent-browser journey verification path that proves the homepage screen inventory and the homepage-to-new-chat transition against a live local instance with isolated runtime state and persisted evidence. The initial slice must stay deterministic by proving screen composition and persisted chat creation without depending on an external provider response.
