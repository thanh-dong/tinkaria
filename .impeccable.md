## Design Context

### Users
Solo developer running Claude Code and Codex on their personal workstation. Sessions often happen during focused evening/weekend hours. Accessed through a browser (including mobile via Cloudflare tunnel), but the primary context is a desktop screen at the developer's own desk.

### Brand Personality
**Quiet, precise, unhurried.** Like a Japanese stationery shop or a Muji store — extreme restraint, invisible design, nothing unnecessary. Every element earns its place through function, not decoration.

### Aesthetic Direction
- **Tone**: Minimal with craft. The restraint IS the design. Whitespace is intentional, not empty.
- **References**: Linear (density without noise), Raycast (snappy, keyboard-driven, dark-first), Muji (invisible design, material honesty)
- **Anti-references**: Notion (too playful/chatty), generic SaaS dashboards (card soup), anything with gradient text or glassmorphism
- **Theme**: Both light and dark. Dark is the primary context (late-hour dev sessions). Light should feel warm and paper-like, not clinical.
- **Brand color**: Warm coral/terracotta — `oklch(71.2% 0.194 13.428)`. Used sparingly as accent, never as fill.

### Design Principles
1. **Silence over noise**: Remove before adding. If removing an element doesn't hurt comprehension, it was never needed.
2. **Data over prose**: Show numbers, timestamps, status. Never write a paragraph where a badge would do.
3. **Earned attention**: The logo coral color appears only when something genuinely needs attention. Overuse kills its power.
4. **Material honesty**: Surfaces feel like surfaces (paper, card, panel), not floating glass. No blur effects, no glow.
5. **Snappy transitions**: Fast ease-out (150-250ms), never bouncy. Motion communicates state change, not personality.
